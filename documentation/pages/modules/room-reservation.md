# Room reservation module

This module takes care of desk bookings.

## Manifest.json

| ----------------------- | ---------------------- |
| id                      | room-reservation       |
| name                    | Meeting Rooms          |
| dependencies            | users                  |
| requiredIntegrations    | []                     |
| recommendedIntegrations | "email-smtp", "matrix" |
